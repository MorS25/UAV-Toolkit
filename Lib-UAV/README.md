# Lib-UAV

Biblioteca em Java que contém estruturas genéricas aos sistemas UAV-MOSA, UAV-IFA, UAV-GCS, UAV-Mission-Creator e UAV-Tests.

Para utilizar essa biblioteca, deve-se importá-la nos sistemas UAV-MOSA, UAV-IFA, UAV-GCS, UAV-Mission-Creator e UAV-Tests.

Neste projeto temos classes que armazenam/controlam diversas informações, como: aeronaves disponíveis (iDroneAlpha e Ararinha), sensores da aeronave (GPS, Atitude, Bateria, Velocidade, etc), dados dos parâmetros, entre outras.

Tem-se classes que gerenciam a aquisição de dados da aeronave através de requisições GET e POST. Tem-se classes genéricas também sobre estrutura de dados genéticas aos sistemas MOSA e IFA. Classes para leituras de arquivos de instâncias, etc. Classes para conversões de coordenadas cartesianas para coordenadas geográficas e vice-versa.
